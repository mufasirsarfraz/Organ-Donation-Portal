# Organ-Donation-Portal
🫀 Excited to share our latest project: ODMS — Organ Donation Management System

As part of our Database Management Systems course, our team built a full-stack web application designed to solve a real, life-critical problem: the lack of digital coordination in organ donation systems.

The Problem:
Manual, paper-based organ donation processes lead to delayed matches, lost records, and preventable loss of life. A donated heart can expire in just 4-6 hours — every minute of delay matters.

What We Built:
A centralized system managing the complete organ donation lifecycle — donor registration, consent verification, organ tracking, recipient matching, transplant surgery, and post-op follow-up — all built on 12 fully normalized database tables.

Key Features:
✅ Live organ viability countdown with automatic expiry alerts
✅ Automated blood group compatibility checks before matching
✅ Consent verification gate — no organ donation without documented consent
✅ Priority-based recipient waitlist sorted by clinical urgency
✅ Real-time status tracking across the entire donation pipeline

Tech Stack:
C# | ASP.NET Core Web API | Entity Framework Core | SQL Server | HTML/CSS/JavaScript

This project taught us far more than database theory — it pushed us to think about real-world clinical workflows, data integrity, and how good database design can genuinely save lives.
